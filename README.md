# react-native-google-nav

React Native Google Navigation SDK wrapper for New Architecture with Expo support

## Installation


```sh
npm install react-native-google-nav
```


## Usage


```js
import { GoogleNavView } from "react-native-google-nav";

// ...

<GoogleNavView color="tomato" />
```
